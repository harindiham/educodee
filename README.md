# EduCode Lecturer Appointment Management System

## Project Description
A web-based Lecturer Appointment Management System developed using React and Firebase. The system allows students to book appointments with lecturers and manage appointment requests.

## Features
- Book appointments
- View appointments
- Appointment management
- Firebase database integration
- Responsive design

## Technologies Used
- React
- Firebase
- Tailwind CSS
- React Router

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/educodee.git
   ```

2. Navigate to the project folder:
   ```bash
   cd educodee
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm run dev
   ```
